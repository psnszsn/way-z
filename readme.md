# Way-Z

Native Zig Wayland implementation

## Description

Wayland client library for Zig apps.
This can be used to crate a statically linked binary capable of displaying CPU rendered graphics.

## Getting Started

### Dependencies

* Zig 0.12.0

### Runnig examples

* `zig build run-hello`
